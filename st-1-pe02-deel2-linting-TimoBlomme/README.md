# CoinChop Application

A .NET-based application for managing expense tracking and group payments, developed as part of the Continuous Integration Advanced course at Howest.

## Project Overview

This project demonstrates modern software development practices including:
- Continuous Integration/Continuous Deployment (CI/CD)
- Automated testing and linting
- GitHub Actions workflows
- Code quality enforcement

## Technology Stack

- **Framework**: .NET 6.0
- **Architecture**: Multi-layer architecture
- **Testing**: xUnit for unit testing
- **CI/CD**: GitHub Actions
- **Linting**: GitHub Super-Linter
